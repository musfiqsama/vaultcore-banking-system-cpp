# 🎯 Exam Management System (C)

A console-based Exam Management System built using C programming language.
This project simulates a real exam environment with login system, timed MCQ exam, admin control, and result analysis.

---

## 🚀 Features

### 👨‍🎓 Student Panel

* Student Signup & Login
* Password masking
* Select exam marks (10 / 20 / 30 / 40 / 50)
* MCQ-based exam
* Live countdown timer
* Negative marking
* Instant result (Score + Percentage + Grade)
* Multiple exam attempts
* Result history
* User profile

---

### 🧑‍💼 Admin Panel

* Admin login
* Add questions
* Edit questions
* Delete questions
* View all questions
* System analytics

---

## 🛠️ Technologies Used

* C Programming
* File Handling
* Console-based UI

---

## ▶️ How to Run

1. Open project in CodeBlocks
2. Add files:

   * `main.c`
   * `functions.c`
   * `exam.h`
3. Build & Run (F9)
